## 品牌分布
- 约翰迪尔：2台收割机 + 1台运粮车 = 3台
- 凯斯：2台收割机 + 1台运粮车 = 3台
- 纽荷兰：1台收割机 + 1台运粮车 = 2台
- 克拉斯：1台收割机 + 1台运粮车 = 2台

## 作业区域分配
- 西北区：约翰迪尔（2收割机 + 1运粮车）
- 东北区：凯斯（2收割机 + 1运粮车）
- 西南区：纽荷兰（1收割机 + 1运粮车）
- 东南区：克拉斯（1收割机 + 1运粮车）

## 作业数据调整
由于设备减少到10台，需要相应调整：
- 日作业面积：约 3,000-5,000 亩/天（10台设备）
- 日收获量：约 2,000-3,000 吨/天
- 秋收周期：约 40-60 天完成 20 万亩
