## 发现的问题汇总

### 🔴 P0 - 严重问题（必须修复）

#### 1. 设备效率百分比超过100%
- **位置**: 左侧设备列表、作业监控页面设备卡片
- **问题**: 效率/健康度显示超过100%（如102%、110%、115%、189%等）
- **影响**: 数据不合理，严重影响专业性
- **修复文件**: `mockData.ts`

#### 2. 运粮车显示"效率"数据
- **位置**: 设备列表中的运粮车卡片
- **问题**: 运粮车不应该显示"效率"百分比，这是收割机专属指标
- **影响**: 业务逻辑错误
- **修复文件**: `CNHSidebar.tsx`

### 🟡 P1 - 中等问题（建议修复）

#### 3. 地图默认加载为普通地图
- **位置**: 机队管理页面地图
- **问题**: 首次加载显示白色背景普通地图，需手动切换卫星地图
- **影响**: 农场场景下卫星地图更实用
- **修复文件**: `Map.tsx`

#### 4. 作业监控设备卡片样式
- **位置**: 作业监控页面
- **问题**: 离线/待命设备卡片与作业中设备区分不够明显
- **修复文件**: `WorkMonitor.tsx`

### 🟢 P2 - 轻微问题（可选修复）

#### 5. 轨迹回放使用Canvas简化图
- **位置**: 轨迹回放页面
- **问题**: 使用Canvas绘制简化轨迹，不如真实地图直观
- **修复文件**: `TrajectoryReplay.tsx`
