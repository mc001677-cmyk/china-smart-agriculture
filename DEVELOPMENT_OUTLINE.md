# 中国智慧农业平台 - 开发大纲与功能规划

## 一、平台定位

本平台面向 **现代化农场 / 合作社负责人 / 农业机手 / 车队管理者 / 二手农机设备经销商**。

集成「卫星地图可视化、农机车队管理、农机健康监测、作业交易、农机设备交易」于一体，帮助管理者做到：

> **“看得见全场、管得住机队、算得清效益、盘得活资源”**

---

## 二、核心功能模块

### 1. 农场经营指挥大屏
*   **地图底座**：采用 **吉林一号卫星底图** + 自定义地块展示整场布局。
*   **分层显示**：支持按管理区、作物类型、作业阶段分层显示。
*   **实时展示**：
    *   当前作业区域
    *   农机分布 & 状态
    *   告警信息与天气情况
*   **核心价值**：一屏上就能看到“今天谁在干活、在哪里干、干得怎样”。

### 2. 农机车队管理与健康监测
*   **机队资产台账**：品牌、型号、马力、购置年份、工作小时数等一目了然。
*   **运行状态监控**（模拟 + 接入真实终端）：
    *   位置、速度、油量、DEF、发动机负荷等运行指标。
*   **健康监测与维保管理**：
    *   发动机油、液压油、滤芯等关键部件健康度（%）。
    *   维保计划、维保记录、费用统计。
*   **决策支持**：支持按设备、品牌、作物、区域做对比分析，帮助决策“哪些车继续用、哪些该淘汰”。

### 3. 智能告警与风险预警
*   **多维告警规则**：
    *   低油量、超速、超长怠速。
    *   维保超期、故障趋势、作业异常。
*   **智能告警中心**：
    *   告警统一汇总。
    *   可按严重程度、设备、区域、时间进行筛选。
    *   支持标记已处理 / 忽略，并保留完整处理记录。
*   **核心价值**：帮助管理者提前发现问题、减少停机时间。

### 4. 作业交易与协同调度平台
*   **适用场景**：大农场 + 周边合作社 / 机手 的协同作业。
*   **作业需求发布**：
    *   支持翻地、播种、施肥、打药、收割、打捆等多种作业类型。
    *   可设置作业面积、作物、时间窗口、作业要求、预算价格等。
*   **机手 / 机队接单**：
    *   支持一口价、竞价两种方式。
*   **全流程跟踪**：待接单 → 进行中 → 待验收 → 已完成。
*   **验收与评价**：
    *   按作业轨迹、覆盖率、作业时长、油耗等指标辅助验收。
    *   双向评价体系：地主评机手、机手评地主。
*   **定位**：可作为内部调度平台，也可逐步对外开放给周边合作社使用。

### 5. 农机设备交易频道（设备资源盘活）
*   **场景**：面向大场与合作社的 **农机买卖 / 调剂** 场景。
*   **设备信息结构化展示**：
    *   **分类**：拖拉机、联合收割机、播种机、施肥机、植保机、插秧机、打捆机、青贮机、装载机、深松机等。
    *   **关键参数**：品牌、型号、年份、小时数、马力、价格、所在地区。
*   **卖家类型**：农场自有、合作社、经销商、服务队。
*   **筛选功能**：支持按设备类型、品牌、价格区间、地区等多条件筛选。
*   **后续扩展**：
    *   设备检测报告
    *   分期 / 金融方案
    *   线上意向登记与电话联系

### 6. 农事记录、历史作业与产量分析（模拟数据已打通）
*   **历史作业轨迹回放**：按日期、设备、地块查看运行轨迹、作业幅宽。
*   **作业统计**：每天、每台机器的作业面积、工时、油耗、效率。
*   **产量与亩产分析（示范版）**：
    *   按地块、作物、机队统计总产量、平均产量。
    *   可对比不同区域、不同作业方案的效果。
*   **核心价值**：为后续接入真实产量监测、精准种植决策提供基础框架。

### 7. 模拟运行 / 正式运行双模式
*   **模拟模式**：
    *   内置一套 20 万亩示范农场的模拟数据。
    *   用于内部演示、培训、方案讨论，不影响正式业务。
*   **正式模式**：
    *   预留接口接入真实农机终端、网关、数据库（MySQL）。
    *   未来可直接对接实际硬件与业务系统。
