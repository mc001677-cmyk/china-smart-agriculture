# 农场收获作业全流程学习笔记

## 一、小麦收获作业全流程

### 1.1 作业前准备

**出车前检查清单：**
- 制动踏板、离合器等操作装置是否正常
- 冷却液及发动机机油是否在正常范围
- 轮胎气压、履带、螺栓、螺母是否松动
- 转速表、指示灯是否异常
- 重要部件检查：凹板筛网、脱粒齿、割刀、扶禾器、拨禾器、分禾器
- 传动链条及传动皮带的灵活性及张紧度

**随车装备：**
- 油料、刀片、皮带、铆钉等常用配件
- 常用工具：扳手、钳子、皮尺、锤子
- 相关证件及机器使用技术资料

### 1.2 收割前技术调试

| 调试项目 | 调试要点 |
|----------|----------|
| **拨禾轮高度** | 弹齿作用在小麦高度的2/3处 |
| **拨禾轮前后** | 向前调试拨禾效果好，向后调试铺放效果好 |
| **弹齿倾角** | 直立作物垂直向下15°，倒伏作物向后倾斜25° |
| **脱粒间隙** | 正常湿度：入口1.6cm，出口0.6cm |
| **送风量** | 湿度大增加送风量，早晚大于中午 |
| **割茬高度** | 全喂入式约12cm |

### 1.3 作业操作要点

1. 起步时保持平稳运行，达到额定转速再开展收割
2. 多采取**顺时针向心回转收割**方式，倒伏作物可逆向收割
3. 地势平坦、谷物成熟可提升速度；湿度大则降低速度
4. 拨禾轮圆周速度是机械前进速度的**1.5倍**
5. 尽可能趋向满幅工作
6. 割茬高度不得小于6cm，留茬约12cm

## 二、玉米收获作业全流程

### 2.1 作业前准备

**重点调整部位：**
1. **发动机**：检查供油时间，气门间隙
2. **切割器总成**：刀头刃口磨损、动刀片齿高、定动刀片间隙
3. **脱粒装置**：脱粒间隙和滚筒转速，左右两边一致

**试运转顺序：**
小油门 → 中油门 → 大油门 → 先局部 → 再全面

### 2.2 玉米收获9大技巧 (来源: No-Till Farmer)

| 技巧 | 设置要点 |
|------|----------|
| **1. 筛子设置** | 底筛全开，上筛关闭以去除碎穗芯 |
| **2. 割台角度** | 直立玉米23-25°，倒伏玉米约20° |
| **3. 聚拢链速度** | 初始55 RPM，出现推土现象则加速 |
| **4. 横向螺旋** | 与托盘间隙1.75英寸，防止籽粒破碎 |
| **5. 剥皮板间隙** | 前端1 1/8英寸，后端1 3/16英寸 |
| **6. 喂入链速度** | 最大长度，最快速度 |
| **7. 滚筒转速** | 增加直到出现破碎籽粒，然后降10 RPM |
| **8. 风扇速度** | 增加直到出现破碎籽粒，然后降10 RPM |
| **9. 导流板设置** | 调至慢速位置减少籽粒损失 |

### 2.3 玉米机收减损要点

- **适宜收获期**：完熟期，中下部叶片变黄，基部叶片干枯
- **损失率标准**：总损失率≤3.5%，籽粒破碎率≤1%
- **66%的损失发生在割台**，需重点调整

## 三、车队联合作业经验

### 3.1 数字化车队管理

**核心功能：**
- **GPS实时追踪**：所有设备位置可视化
- **田间物流**：卡车数量、等待时间、重新调度
- **进度追踪**：已收获面积、下一地块预估
- **设备监控**：远程查看设备参数，指导操作

**关键价值：**
- 减少停机时间
- 减少无效沟通（电话、对讲机）
- 快速决策，节省燃油和人工

### 3.2 约翰迪尔 Machine Sync 技术

**核心功能：**
- 收割机与运粮车之间建立**无线网络连接**
- 拖拉机位置和速度与收割机**自动同步**
- 自动引导运粮车到预设"home point"进行卸粮

**关键优势：**

| 优势 | 效果 |
|------|------|
| 提高生产效率 | 边走边卸提升8.3%效率 |
| 减少粮食洒落 | 精确同步减少溢出 |
| 提高安全性 | 降低碰撞风险，尤其夜间/扬尘条件 |
| 减少土壤压实 | 优化行驶路径 |
| 支持多机协同 | 最多6台设备共享位置和导航线 |

**工作原理：**
1. 收割机周围显示操作区域（橙色线）
2. 运粮车进入操作区域后启动同步
3. 拖拉机自动跟随收割机移动，无需操作员干预
4. 可调整默认操作区域大小

### 3.3 收获物流配置建议

| 设备 | 建议配置 |
|------|----------|
| **运粮车** | 750-1000蒲式耳，够1-2次收割机卸粮 |
| **卡车** | 每小时2-3车次，近距离1台，远距离2台 |
| **协调原则** | 收割机减速比加速容易，保持螺旋在车斗上方 |

## 四、收割机-拖拉机-运粮车协同作业

### 4.1 边走边卸 (Unloading on the Go)

**操作要点：**
- 收割机操作员负责保持卸粮螺旋在运粮车上方
- 运粮车匹配收割机速度和方向
- 收割机减速比加速更容易控制

**自动化技术：**
- **Raven Cart Automation**：175m×55m范围内自动同步
- **Case IH Cart Automation**：自动同步方向和速度
- **John Deere Machine Sync**：无线连接，自动跟随

### 4.2 中国农机协同作业研究

**多机协同调度目标：**
- 总非生产性作业时间最短
- 运粮车辅助收割机协同卸粮
- 收获机仓满时有序等待运粮车(HWU模式)
- 作业中无需停车即可卸粮(TFU模式)

**双卸粮协同系统效果：**
- 作业时间减少13.58%
- 保证作业流畅性

### 4.3 跨区作业调度

**河南"三夏"经验：**
- 全省农机保有量和作业需求量详细摸底
- 发布机收作业需求和机具供给动态信息
- 引导收获机械有序流动
- 高速公路开通"绿色通道"保障农机运输

## 五、设备使用经验总结

### 5.1 收割机日常保养

| 保养项目 | 周期/时机 |
|----------|-----------|
| 散热器清洗 | 每日/堵塞时 |
| 滤清器清洗 | 定期/冒黑烟时 |
| 刀具检查 | 每日 |
| 皮带张力 | 每周 |
| 链条润滑 | 每周 |
| 焊接件检查 | 作业前 |

### 5.2 常见故障处理

| 故障 | 原因 | 解决方法 |
|------|------|----------|
| 刀具损坏 | 铆钉松动、高低差大 | 调整护刃器，拧紧铆钉 |
| 推运器打滑 | 叶片磨损 | 锉刀修复或更换 |
| 滚筒堵塞 | 喂入量过大 | 改装滚筒，更换芯 |
| 筛面堵塞 | 张开度不够 | 增加送风量 |
| 脱粒不净 | 间隙大、转速低 | 缩小间隙，提高转速 |
| 链条掉链 | 松紧不当 | 调整松紧度 |

### 5.3 存放保养

- 清除残留物，清水清洗外部
- 摩擦部位涂抹润滑油防锈
- 每月运转一次发动机
- 卸下传动皮带涂滑石粉
- 割台下降至水平位置，用木料垫起
- 轮胎用千斤顶顶起，避免变形
- 油箱柴油放净并密封
