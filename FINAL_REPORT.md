## 一、系统概述

### 1.1 项目背景

友谊农场位于黑龙江省双鸭山市，是中国最大的国有农场之一。本平台为其20万亩玉米秋收作业提供智能化管理支持，实现对80台多品牌农机的实时监控、调度和数据分析。

### 1.2 技术架构

| 层级 | 技术栈 |
|------|--------|
| **前端** | Vite + React 19 + TypeScript + TailwindCSS + Radix UI |
| **后端** | Node.js + Express + Drizzle ORM |
| **数据库** | MySQL (TiDB) |
| **地图服务** | 高德地图 (AMap) |
| **设计风格** | Apple-style 玻璃态设计 + 毛玻璃效果 |
